## 프로젝트 개요 (Korean)

PixelOS는 32비트 x86 환경에서 동작하는 커스텀 운영체제입니다. </p>
핸드크래프트 부트로더, 작은 C 커널, 간단한 그래픽 시스템을 갖추고 있으며, OS 위에서 바로 게임을 실행할 수 있도록 개발되었습니다.

- 저수준 프로그래밍 경험
- 운영체제 설계 이해
- 게임 엔진 개발 경험

## Introduction (English)

PixelOS is a fully custom 32-bit x86 operating system with a handcrafted bootloader, minimal C kernel, and simple graphics subsystem. </p>
It can run a game directly on the OS, demonstrating:

- Low-level programming
- OS design
- Game engine development

## Features / 특징

- Bare-metal 32-bit x86 development / 베어메탈 32비트 x86 개발
- Minimal OS kernel with basic multitasking / 최소 커널과 간단한 멀티태스킹
- Simple graphics rendering and input handling / 그래픽 렌더링 및 입력 처리
- Custom game running without underlying OS / OS 없이 직접 실행되는 게임
