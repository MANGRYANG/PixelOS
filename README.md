## 프로젝트 개요

PixelOS는 32비트 x86 환경에서 동작하는 커스텀 운영체제입니다. </p>
핸드크래프트 부트로더, 작은 C 커널, 간단한 그래픽 시스템을 갖추고 있으며, OS 위에서 바로 게임을 실행할 수 있도록 개발되었습니다.

- 저수준 프로그래밍 경험
- 운영체제 설계 이해
- 게임 엔진 개발 경험

## 주요 기능

- 베어메탈 32비트 x86 개발
- 최소 커널과 간단한 멀티태스킹
- 그래픽 렌더링 및 입력 처리
- 간단한 커스텀 게임 실행

## 사용 기술

- NASM (어셈블리)
- C (커널)
- x86 아키텍처 이해 및 하드웨어 제어
