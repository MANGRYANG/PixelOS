## 🧭 프로젝트 개요

본 프로젝트는 하드웨어 및 운영체제 자원을 직접 이해하고, 게임이 OS 위에서 어떻게 동작하는지 탐구하는 것을 목표로 진행되었습니다.  
주요 목표는 다음과 같습니다.

- 저수준 프로그래밍 능력 향상
- 운영체제 설계 및 구조 이해
- 게임과 운영체제 간 상호작용 이해

---

## 🪄 주요 기능

PixelOS를 통해 구현하고 경험한 핵심 기능은 다음과 같습니다.

- 부트로더 및 커널 초기화
- 메모리 및 시스템 관리
- 그래픽 및 입력 처리
- 게임 루프 구현
- 성능 분석 및 최적화

---

## 🔧 사용 기술

- NASM (어셈블리) : 부트로더
- C : 커널 및 게임 프로그램
